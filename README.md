# 🌿 Vida Leve — Psicologia & Bem-Estar

> Site institucional fictício desenvolvido como projeto acadêmico,
> representando uma clínica de psicologia focada em saúde mental e qualidade de vida.

---

## 📋 Sobre o projeto

O **Vida Leve** é um site de 5 páginas criado do zero com HTML, CSS e JavaScript.

O objetivo foi desenvolver uma identidade visual acolhedora e profissional para uma
clínica fictícia de psicologia, praticando conceitos de layout, responsividade
e organização de código front-end.

---

## 🛠️ Tecnologias utilizadas

- HTML5 semântico
- CSS3 (variáveis, flexbox, media queries)
- JavaScript vanilla

---

## 📄 Páginas

| Página       | Descrição                              |
|--------------|----------------------------------------|
| `index.html` | Home — apresentação da clínica         |
| `sobre.html` | Sobre a equipe e a missão              |
| `servicos.html` | Serviços e especialidades oferecidos |
| `blog.html`  | Artigos e conteúdos de bem-estar       |
| `contato.html` | Formulário e informações de contato  |

---

## 🎨 Design

- **Paleta:** tons de vinho, rosé e branco
- **Tipografia:** Playfair Display (títulos) + Lato (corpo)
- **Estilo:** clean, acolhedor e profissional
